# Math Coaster 3D

Welcome to **Math Coaster 3D**! This is a web-based procedural 3D roller coaster game. The tracks are dynamically generated based on complex mathematical equations (like Lissajous curves, Trefoil Knots, and Figure 8s), and you can ride them in a first-person POV mode!

## 🚀 Getting Started

This application is built as a highly performant, stateless web application utilizing modern WebGL via Three.js and Vite as the bundler.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Navigate to the project directory:
   ```bash
   cd math_coaster_3d
   ```

2. Install the necessary dependencies (Three.js and Vite):
   ```bash
   npm install
   ```

### Running the Local Server

To play the game and see the 3D graphics in action, start the local development server:

```bash
npm run dev
```

Your terminal will display a local URL (usually `http://localhost:5173/`). Simply open this URL in any modern web browser to start the game!

## 🎢 Controls & Features
- **Track Equation:** Use the dropdown to select different mathematical track algorithms. 
- **Ride POV:** Click this button to lock onto the track and experience the coaster ride from a first-person view.
- **Ride Speed:** Dynamically change the speed of your cart while riding.
- **Track Color:** Customize the glowing neon track by picking your favorite hex color at runtime!

Enjoy the ride!
